# Ableton Extensions SDK (local install)

This example depends on packages **not** published to public npm.

1. Download `@ableton-extensions/sdk` and `@ableton-extensions/cli` from [Ableton's Extensions SDK](https://ableton.github.io/extensions-sdk/).
2. Install into this folder:

```bash
npm install /path/to/ableton-extensions-sdk-1.0.0-beta.0.tgz
npm install -D /path/to/ableton-extensions-cli-1.0.0-beta.0.tgz
```

See also [`vendor/README.md`](../../vendor/README.md) at the repo root.
