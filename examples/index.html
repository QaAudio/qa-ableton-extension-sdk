QuantumAudio Extension SDK examples

Overview Buttons Toolbar Sliders Dialog Parameter panel Theme toggle Select Vue layouts

Static HTML demos using Ableton Live styling from the official Extensions SDK tokens. For copy-paste Vue layouts (toolbar, dialogs, effect panels), run npm run dev:examples or open examples/vue.

Quick preview

Pages

  • Buttons — pill buttons, groups, pressed state
  • Sliders — horizontal range controls with value readout
  • Dialog — modal-style form like the SDK rename clip example
  • Parameter panel — grouped labels, sliders, and actions
  • Theme toggle — switch between Live dark and light tokens
  • Select — native chooser styling (knobs/faders/meters are Vue-only)
  • Vue layouts — real-world gallery (Vite; npm run dev:examples)
  • Starter extension — full Ableton project (Tempo Tap; plain HTML, no Vue)
